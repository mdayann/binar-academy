Tutorial Git

  1. Pertama,pada terminal dalam direktori ketik perintah git status untuk melihat status repositorinya
  2. Jika ada file yang berubah dan belum ditambah ke git ketik git add 'nama_file' atau bisa git add . untuk menambah semua file ke git.
  3. Setelah itu, perlu kita commited agar semua perubahan disimpan oleh Git dengan cara git commit -m "komen" .
  4. Setelah itu, kita kirim ke repositori online github dengan cara git push origin master .
